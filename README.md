# FUTURE_PE_02
# AI Content Marketing using UGC Ads


## Project Overview

This project presents a complete AI-generated UGC (User-Generated Content) advertising content pack for Brew Haven Café. It demonstrates how prompt engineering and AI content marketing can be used to create high-conversion short-form video ad assets for social media platforms.

The content is designed for direct use in marketing campaigns, influencer-style videos, and paid ads across Instagram, Facebook, and YouTube Shorts.

## Prompt Strategy

The prompt was structured to guide the AI in generating high-quality website copy by:

* Assigning the role of an expert website copywriter
* Defining business type, audience, and tone clearly
* Focusing on café experience, cravings, and comfort
* Generating value-driven homepage content
* Writing simple and appealing service descriptions
* Creating strong, action-oriented CTAs
* Ensuring a warm and inviting brand voice

This structured approach ensures consistent, realistic, and conversion-focused output.
